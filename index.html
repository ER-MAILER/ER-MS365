Frequently Asked Questions

Find answers to common questions about our software

Do I need a Microsoft Azure account to use this software?

Yes, you need an active Microsoft Azure subscription to upload data to Azure cloud services. Our software acts as a bridge between your local files and Azure storage.

What Azure services are supported?

Our software supports Azure Blob Storage, Azure Files, and Azure Data Lake Storage. Premium edition includes additional support for custom Azure integrations.

Is this a one-time payment or subscription?

Both Standard and Premium editions are available with a one-time payment. You pay once and get the software with updates included (1 year for Standard, lifetime for Premium).

What operating systems are supported?

Currently, our software is compatible with Windows 10 and Windows 11. We're working on macOS and Linux versions for future releases.

Do you offer technical support?

Yes, we offer email support for all our customers. Premium users receive priority support with phone and faster response times.

Need Technical Support?

Our team is available to help you with installation, configuration, and any technical issues
